To setup

  1. Download the source code as zip from GitHub. Alternatively, use git clone.
  2. Create a new branch of this repository using git branch.
  3. Make modification there and push to the your branch.
  4. I will merge everything to the master branch after everything is tested working.

To run

  1. Open XAMPP, start Apache and MySQL.
  2. Copy the source folder "EventWeb" into C:\xampp\htdocs\
  3. Go to web browser, run localhost/EventWeb/config/database.php first.
  4. Then, run localhost/EventWeb/home.php.

To make modification

  1. To create a new file, follow the naming convention: filename.php, where filename is using camel case, such as resetPassword, instead of using a space ("reset password").
  2. Copy the content in template/template.php into your new file. It contains boilerplate for a new file.
  3. Follow the instructions in that file.
  4. To add the new file into the header (navbar), refer to partials/header.php.

Importing files

  1. For all master style files, add into the partials/html-header.php under import comment. Otherwise the file can be added after the import comment.
  2. For all master script files, add into the partials/footer.php under import comment. Otherwise, the file can be added after the import comment.
